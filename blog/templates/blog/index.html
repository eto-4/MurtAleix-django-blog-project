Projecte Blog

Benvingut al meu projecte personal desenvolupat amb Django. Aquest blog serveix com a pràctica per aprendre sobre el desenvolupament web amb Python, la gestió de dades i la creació d’interfícies amigables amb HTML, CSS i Bootstrap.

Objectius del projecte

Aquest projecte té com a finalitat explorar el funcionament intern d’un blog, des de la seva estructura fins a la manera com es mostren les dades. A més, busca aplicar coneixements reals de backend i frontend, així com millorar la comprensió del treball amb bases de dades, models i formularis.

Tecnologies utilitzades

  • Django — per a la creació del backend i gestió del servidor.
  • HTML & CSS — per estructurar i donar estil a les pàgines.
  • Bootstrap — per a una estètica responsiva i ràpida.
  • SQLite — com a base de dades per guardar posts, autors i etiquetes.
